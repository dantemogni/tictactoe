### First steps
#### Install all dependencies needed 
```bash
make install
```
### Go live
#### Start the server
```bash
make serve
```
_Will start listening at port `2000`_

#### Start client
In another terminal...

```bash
make start-client
```
_Will start frontend app in port `3000`_
